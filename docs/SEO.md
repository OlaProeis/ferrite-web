# SEO Configuration

This document describes the SEO setup for getferrite.dev.

## Overview

The site uses Nuxt 3's built-in SEO capabilities along with several enhancements for search engine visibility.

## Implemented Features

### Meta Tags (`nuxt.config.ts`)

- **Title & Description** - Site-wide defaults
- **Open Graph** - Facebook, LinkedIn sharing
- **Twitter Cards** - Twitter sharing with large image
- **Theme Color** - Browser UI coloring
- **Robots** - Indexing directives for search engines

### Structured Data (JSON-LD)

Located in `app.vue`:

| Schema Type | Purpose |
|-------------|---------|
| `SoftwareApplication` | Identifies Ferrite as downloadable software |
| `WebSite` | Basic site identity |
| `Organization` | Brand recognition, social links |

Page-specific schemas:

| Page | Schema |
|------|--------|
| `/faq` | `FAQPage` - Enables FAQ rich snippets |
| `/blog/*` | `Article` - Blog post metadata |
| All pages | `BreadcrumbList` - Navigation breadcrumbs |

### Sitemap

Auto-generated by `@nuxtjs/sitemap` module.

- URL: https://getferrite.dev/sitemap.xml
- Includes all pages and blog posts
- Referenced in `robots.txt`

### RSS Feed

- URL: https://getferrite.dev/rss.xml
- Auto-discovered via `<link rel="alternate">` tag
- Generated by `server/routes/rss.xml.ts`

### Canonical URLs

Handled by `plugins/seo.client.ts` - sets canonical URL on each page navigation.

### Composables (`composables/useSeo.ts`)

Reusable SEO utilities:

```typescript
// Set canonical URL
useCanonicalUrl()

// Add breadcrumb schema
useBreadcrumbs([
  { name: 'Home', path: '/' },
  { name: 'Features' },
])

// Get absolute URL
const url = getAbsoluteUrl('/download')

// Get site URL constant
const siteUrl = useSiteUrl()
```

## Search Console Setup

### Google Search Console

1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Add property: `https://getferrite.dev`
3. Verify via DNS TXT record (add in Vercel DNS settings)
4. Submit sitemap: `https://getferrite.dev/sitemap.xml`

### Bing Webmaster Tools

1. Go to [bing.com/webmasters](https://www.bing.com/webmasters)
2. Import from Google Search Console

## Testing Tools

## File Reference

| File | SEO Purpose |
|------|-------------|
| `nuxt.config.ts` | Global meta tags, sitemap config |
| `app.vue` | Site-wide structured data schemas |
| `composables/useSeo.ts` | Reusable SEO utilities |
| `plugins/seo.client.ts` | Canonical URL handling |
| `public/robots.txt` | Crawler directives |
| `server/routes/rss.xml.ts` | RSS feed generation |

## Page-Level SEO

Each page should include:

```vue
<script setup lang="ts">
useSeoMeta({
  title: 'Page Title - Ferrite',
  description: 'Page description for search results.',
  ogTitle: 'Page Title',
  ogDescription: 'Description for social sharing.',
  ogImage: 'https://getferrite.dev/img/og-image.png',
})

useBreadcrumbs([
  { name: 'Home', path: '/' },
  { name: 'Current Page' },
])
</script>
```
